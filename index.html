MooEditable by cheeaun

A simple web-based WYSIWYG editor, written in MooTools.

Demo

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/cheeaun/mooeditable

Build

You can build a single compressed JavaScript file and the save it, using the MooEditable Builder.
